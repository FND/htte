命令行

全局选项

  • config: 项目配置文件路径，默认 htte.yaml

run

运行测试, 是默认命令。

选项:

  • debug: 打印请求和响应数据　
  • amend: 从上次中断的地方开始运行测试
  • bail: 测试不通过，则中断后续测试
  • unit: 从指定的单元测试处开始运行测试
  • shot: 执行单个测试，不再运行后续测试

分享一些实用技巧：

  • 某个测试失败时暂停执行
  • 从上次失败的地方开始执行, 适合接口调试
  • 仅运行指定测试
  • 显示调试信息，将同步打印请求响应数据

view

查看测试, 可以看到测试组织结构。

选项:

  • module: 通过模块名筛选
  • api: 通过 API 筛选

inspect

审视单元测试，显示该测试一切信息
