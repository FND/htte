Htte

描述式 HTTP 接口自动测试框架。

npm install -g htte
Star Fork

  • 描述式

    采用 YAML 编写测试，上手简单，开发难度低，与后端代码无耦合。

  • 简明

    接口测试被简化成描述请求和响应，Htte 后台自动进行请求生成发送，响应比对断言，测试结果打印。

  • 开源

    Htte 使用 MIT 许可证，100% 开源，Github 可见全部代码。

  • 可扩展

    提供插件自定义请求数据生成和响应数据断言, 允许扩展编码类型以应对多种多样的 Content-Type。
